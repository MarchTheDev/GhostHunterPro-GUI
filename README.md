# Ghost Hunter Pro

Ghost Hunter Pro is a desktop tool that helps you find and clean leftover game files.

It is designed to make common leftover folders easier to inspect, organize, archive, and remove.

## Main features
- Search a game by **name** or **Steam AppID**
- View known leftover paths for a specific game
- Browse a **Library** of detected leftovers found on the PC
- Archive games you do not want to see in the active library
- Open leftover folders directly in Explorer
- Delete selected leftovers or all leftovers for a game
- Keep a small **Recent Hunts** history

## Sections
### Home
Use the search bar to hunt leftovers for a specific game.

### Library
Browse detected leftover games in a visual grid and manage them more easily.

### Settings
Check version/build information and manage update-related actions.

## Notes
- Always review paths before deleting anything.
- Some actions may work better when run with Administrator permissions.
- The app may detect both cracked leftovers and legitimate Steam-owned traces depending on where files are stored.
